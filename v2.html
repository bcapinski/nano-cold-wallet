About

Nano Cold Wallet is a web-based light wallet that allows offline signing of transactions. This allows you to keep your seed on an air-gapped computer, and only use it when you need it. Your seed will never touch a computer that's connected to the Internet, dropping possible attack vectors to nearly zero.

For collecting the data needed to create a state block and then broadcasting it to the network, I use the Nanode Node API. Unforunately, I can't afford to use my own API key, so you'll have to sign up (it's easy), and fill your key in to the fields where the wallet asks for it. This elminates most node issues my own node would have, and allows you to independantly verify the data is correct.

Instructions

  • Go to the hot wallet and input your API key, your address, the destination address, and amount to send. Copy the information to your airgapped computer either manually or with a flash drive.
  • Download the website onto your computer where your seed is stored and input the required information. This requires a private key, so you can use the tools site to generate it from your seed. Copy the outputed block to your hot computer.
  • Input the block into the send transaction form along with your API key. It will broadcast your block to the network.

Hot Wallet

Requires Nanode API key. A state block requires: Private Key, Proof of Work, Previous Hash, Representative Address, Resulting Balance, and Destination Address.
