Nano Cold Wallet

Nano Cold Wallet is a web-based light wallet that allows offline signing of transactions. This allows you to keep your seed on an air-gapped computer, and only use it when you need it. Your seed will never touch a computer that's connected to the Internet, dropping possible attack vectors to nearly zero.

For collecting the data needed to create a state block and then broadcasting it to the network, I use the Nanode Node API. Unfortuunately, I can't afford to use my own API key, so you'll have to sign up (it's easy), and fill your key in to the fields where the wallet asks for it. This elminates most node issues my own node would have, and allows you to independantly verify the data is correct.
